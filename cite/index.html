How to cite Brian

In a paper

When you use Brian in your research, please cite our main paper:

Stimberg M, Brette R, Goodman DFM (2019)
Brian 2, an intuitive and efficient neural simulator
eLife 8:e47314. doi: 10.7554/eLife.47314

BibTeX (click to expand)
@article{Stimberg2019,
    title = {Brian 2, an intuitive and efficient neural simulator},
    volume = {8},
    issn = {2050-084X},
    doi = {10.7554/eLife.47314},
    journal = {eLife},
    author = {Stimberg, Marcel and Brette, Romain and Goodman, Dan FM},
    editor = {Skinner, Frances K},
    month = aug,
    year = {2019},
    pages = {e47314}
}


Some journals use an RRID to reference software tools. The Brian simulator is registered as SCR_002998.
