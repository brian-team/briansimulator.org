Brian 2.0rc3

Brian team

2016-06-16

Source

We are happy to announce the release of Brian 2.0rc3. This release fixes several bugs, in particular related to long-running simulations (a memory leak and the inability to configure the logging system). It  also improves the random number generation mechanism (same high-quality random numbers in C++ standalone as in runtime mode; new seed function to get reproducible random numbers). See the release notes for more information and other changes.

How to get Brian 2: follow the installation instructions in the documentation

Further information about Brian2: http://brian2.readthedocs.org

As always, please report bugs or suggestions to the github bug tracker or to the brian-development mailing list (brian-development@googlegroups.com).
